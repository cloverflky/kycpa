🔍 考点一：混合成本的分解 & 变动成本法

能力等级：★★ (2016,2019-2025历年考核)  |  大纲要求：混合成本分解、变动成本法(七、管理会计·本量利一般关系)

📐 1. 混合成本分解方法

账户分析法简单粗糙，根据会计账户特征划分。
工业工程法无历史数据或验证时使用，标准成本制定更科学。
直线回归法∑Y = na + b∑X ; ∑XY = a∑X + b∑X²
高低点法
单位变动成本 b = (高点成本-低点成本)/(高点业务量-低点业务量)
固定成本 a = 高点(低点)总成本 - b×高点(低点)业务量
💡 总成本线性模型： Y = a + bX   (固定成本 + 变动成本)

📘 2. 变动成本法 vs 完全成本法

项目变动成本法完全成本法
产品成本直接材料+直接人工+变动制造费用直接材料+直接人工+变动制造费用+固定制造费用
期间费用固定制造费用+销管费用销管费用
利润关系完全成本法利润 = 变动成本法利润 + 期末存货固定制造费用 - 期初存货固定制造费用
✅ 学霸笔记： 本量利分析中的“变动成本”是广义的——包括变动生产成本+变动销售管理费用。公式中边际贡献按产品边际贡献（总营业边际贡献）计算。

🧮 考点二：本量利分析基本模型

四大假设： 相关范围假设 | 模型线性假设(单价、单位变动成本常数) | 产销平衡假设 | 品种结构不变假设(多品种)

📌 核心损益方程式：
息税前利润(EBIT) = 单价×销量 - 单位变动成本×销量 - 固定成本
= (单价 - 单位变动成本) × 销量 - 固定成本 = 边际贡献 - 固定成本

📈 边际贡献方程式

  • 边际贡献 = 销售收入 - 变动成本 = 销量 × 单位边际贡献
  • 制造边际贡献 = 销售收入 - 变动生产成本   |   产品边际贡献 = 制造边际贡献 - 变动销售管理费用
  • 边际贡献率 = 边际贡献/销售收入 = 单位边际贡献/单价 ; 变动成本率 + 边际贡献率 = 1
  • EBIT = 销售收入 × 边际贡献率 - 固定成本
  • 多品种加权平均边际贡献率 = Σ(各产品边际贡献率 × 销售比重)

🎯 考点三：保本分析（盈亏临界点）★★★

保本量
保本量 = 固定成本 / (单价 - 单位变动成本) = 固定成本 / 单位边际贡献
保本额
保本额 = 固定成本 / 边际贡献率

📏 安全边际与利润

  • 安全边际量 = 实际(预计)销售量 - 保本量   |   安全边际额 = 实际销售额 - 保本额
  • 安全边际率 = 安全边际量(额) / 实际销售量(额) | 盈亏临界点作业率 + 安全边际率 = 1
  • 息税前利润 = 安全边际量 × 单位边际贡献 = 安全边际额 × 边际贡献率
  • 销售息税前利润率 = 安全边际率 × 边际贡献率
💡 多品种保本分析： 加权平均保本销售额 = 固定成本总额 / 加权平均边际贡献率；某产品保本额 = 加权平均保本额 × 该产品销售占比。

🏆 考点四：保利分析（目标利润）

🎯 保利量 = (固定成本 + 目标利润) / 单位边际贡献
🎯 保利额 = (固定成本 + 目标利润) / 边际贡献率

适用于单一产品或多品种条件下的目标销量/销售额规划。

⚡ 考点五：利润敏感分析 ★★

核心： 各因素(单价、单位变动成本、销量、固定成本)变动引起利润变化的敏感程度。盈亏临界点求各参数极值。

📐 敏感系数 = 目标值变动百分比 / 参数变动百分比
🧩 具体简化公式（只变动一个因素时）：
• 单价的敏感系数 = 基期营业收入 / 基期EBIT  =  P×Q / EBIT
• 单位变动成本的敏感系数 = - 基期变动成本总额 / EBIT
• 销量的敏感系数 = 基期边际贡献 / EBIT = 经营杠杆系数
• 固定成本的敏感系数 = - 基期固定成本 / EBIT
⚠️ 绝对值＞1为敏感因素，正负表示同向/反向变动。
🔥 高频结论： 单价敏感系数最高（绝对值最大）；销量敏感系数 = 经营杠杆系数；安全边际率 = 1 / 经营杠杆系数。

📑 核心公式汇总 & 衍生关系

▸ 基本方程式
EBIT = (P-V)×Q - F
EBIT = 边际贡献 - F
▸ 保本点
Q0 = F / (P-V)
S0 = F / 边际贡献率
▸ 安全边际率
MSR = (Q实际-Q0)/Q实际
EBIT = 安全边际额 × 边际贡献率
▸ 保利量/额
Q目标 = (F+π)/ (P-V)
S目标 = (F+π) / 边际贡献率
▸ 多品种加权边际贡献率
WACMR = Σ (CMRi × 销售比重i)
综合保本额 = F / WACMR
▸ 敏感系数关系
销量敏感系数 = DOL = 边际贡献/EBIT
安全边际率 = 1/DOL

⚠️ 本章重难点 & 新增点说明

📌 2025年考纲/教材调整
✅ 新增/调整内容： 根据2025考纲及教材变化，原“按成本性态分类”内容移至第十二章《成本管理会计的基础概念和基本理论》。本章聚焦混合成本分解、变动成本法、本量利核心模型。同时2025大纲在“本量利一般关系”中明确混合成本分解、变动成本法以及利润敏感分析的能力等级(2~3)。
📖 特别关注： 变动成本法与完全成本法下利润差异（存货吸收固定制造费用）为高频计算题；多品种保本分析与保利分析结合收入比重变化；利润敏感系数的综合应用(结合安全边际率/经营杠杆)。
⭐ 难点一： 完全成本法与变动成本法利润差异调节，尤其是期末/期初存货固定制造费用的计算，先进先出法下对利润的影响。
⭐ 难点二： 混合成本分解中高低点法与回归法的对比，以及延期变动成本与半变动成本图形判别（固定部分是否包含业务量门槛）。
⭐ 难点三： 加权平均边际贡献率在多品种保本额计算，以及安全边际率与杠杆系数之间的转换（客观题常见陷阱）。
⭐ 必考综合点： 本量利分析与短期经营决策（特殊订单定价、自制外购）、资本预算中经营杠杆、以及责任会计中的内部转移定价结合命题。

📌 真题关键模型提炼（基于BT+教材例题）

  • 【高低点法预测】 选择业务量最高最低点，求b = Δ成本/Δ工时，再计算固定成本，代入预计业务量 → 2024年机修成本测算类似题。
  • 【变动成本法与完全成本法利润调整】 差异 = 期末存货所含固定制造费用 - 期初存货固定制造费用，先进先出法需注意单位成本。
  • 【多品种边际贡献及保利门票提价】 联票收入分成50%，注意普通票与优惠票比例，涉及变动管理费(收入的10%)，层层边际贡献计算后解方程。
  • 【敏感系数及安全边际率】 利用“安全边际率=1/经营杠杆系数”，以及盈亏临界点销量 = 当前销量×(1-1/敏感系数)的灵活推导。

🧠 彬哥学霸笔记·应试技巧

易混淆点区分技巧
半变动成本 vs 延期变动成本半变动成本基础固定部分不随业务量变化，延期变动在零点后跃升（有固定部分包含业务量门槛）
变动成本法下产品成本仅包括变动制造成本固定制造费用全额计入期间费用；完全成本法下固定制造费用在产品存货中递延
边际贡献（未定语）= 产品边际贡献= 销售收入 - 变动生产成本 - 变动销售管理费用
安全边际率与利润敏感性敏感系数绝对值越大越敏感，销量敏感系数即为DOL，安全边际率×边际贡献率=销售息税前利润率
